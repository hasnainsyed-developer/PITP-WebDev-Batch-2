⭐ HTML (HyperText Markup Language)

HTML is the basic building block of every website.
It provides the structure of a webpage, just like bones in the human body.

With HTML we can:

Add headings, paragraphs, images, links
Create lists, tables, forms
Organize the content properly

HTML is the first step of web development and it defines what will appear on the screen.

⭐ CSS (Cascading Style Sheets)

CSS is used for styling and designing the website.
It decides how the website will look.

CSS helps in:

Adding colors
Changing fonts
Designing layouts
Creating animations
Making the website responsive

Without CSS, every website would look plain and simple. CSS adds beauty and life to webpages.

⭐ JavaScript (JS)

JavaScript is used to make the website interactive and dynamic.
It controls the behavior of the webpage.

JavaScript is used for:

Showing and hiding content
Creating sliders and animations
Validating forms
Making interactive menus
Building modern web apps

JS makes the website active and smart.
